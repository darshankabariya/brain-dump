# ZK and Blockchain

Notes on zero-knowledge proofs, Ethereum rollups, and supporting L2 infrastructure.

## Layout

- **[zk-fundamentals/](./zk-fundamentals/)** — ZK cryptography and proof systems (SNARKs, STARKs), research, glossary. Start at [`START-HERE.md`](./zk-fundamentals/START-HERE.md).
- **[rollups/](./rollups/)** — Ethereum L2 rollups, architecture, and L1/L2 interactions. See [`ethereum-rollup-interactions.md`](./rollups/ethereum-rollup-interactions.md).
- **[infrastructure/](./infrastructure/)** — Notes on EigenLayer, NEAR, and Succinct SP1.
- **[resources/](./resources/)** — PDFs and reference material (e.g. MoonMath prerequisites).

## Distinction

- `zk-fundamentals/` → ZK cryptography and proof theory.
- `rollups/` → Ethereum scaling and rollup architecture (which happens to use ZK proofs).
